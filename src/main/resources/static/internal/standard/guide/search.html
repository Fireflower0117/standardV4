기본 검색

1. 검색항목은 기본 4개 기준으로 배열하고 width에 따라 2~4열로 수정한다.
2. colgroup은 col w90px / col / col w120px / col / col w120px / col / col w120px / col 8개로 동일하게 적용한다.
3. input은 기본 100%로 적용된다.
4. 검색폼 상하단 여백은 12px (4, 8의배수로 활용)
5. input과 버튼크기는 height : 32px, font-size : 13px로 공통이고, 버튼너비는 80px로 통일한다.

// search detail
$(document).ready(function () {
    $(".search_basic.detail .btn_detail").click(function () {
        if ($(this).hasClass("on")) {
            $(".search_basic.detail .btn_detail").removeClass("on");
            $(".js_srchDet").stop().slideUp(200);
        } else {
            $(".search_basic.detail .btn_detail").removeClass("on");
            $(".js_srchDet").stop().slideUp(200);
            $(this).addClass("on").siblings(".js_srchDet").stop().slideDown(200);
        }
        return false;
    });
});
