모바일 기본

  • 설명
  • 파일

설명

1. html에 meta 태그 추가
html파일 <head></head> 내부에 <meta name="viewport" content="width=device-width, initial-scale=1" /> 추가
2. 단위
└ rem 사용
└ html의 font-size = 1rem (ex. html {font-size:10px} 인 경우 1rem = 10px)
└ reset.css에 반드시 html의 font-size가 선언 되어 있어야 함.
└ 가장 작은 화면 기준으로 단위 환산. 태블릿 버전에서는 기준 단위인 html의 font-size를 키움으로써 자연스럽게 크기 조절
└ ex) 시안이 720px 기준으로 제작된 경우, 360px 화면에서는 모든 단위가 1/2로 줄어듬. 720px 화면에서 font-size가 30px인 텍스트가 360px 화면에서는 15px. 모바일 버전에서는 html의 font-size가 10px 이므로 15/10 = 1.5rem
3. media query 분기
└ 반응형 사이트의 경우, 아래의 3가지 분기점을 사용한다.
  • css
/* pc버전 스타일 */
html {font-size: 10px;}
@media screen and (max-width: 1024px) { 
    /* 태블릿 버전 스타일 (모바일~태블릿 공통 스타일) */ 
    html {font-size: 12px;}
}
@media screen and (max-width: 767px) { 
    /* 모바일 버전 스타일 (태블릿 대비 모바일에서 달라지는 스타일) */
    html {font-size: 11px;}
}
@media screen and (max-width: 480px) { 
    /* 모바일 버전 스타일2 (모바일 작은 화면에서 달라지는 스타일) */
    html {font-size: 10px;}
}

└ 모바일 only 사이트의 경우, 아래의 2가지 분기점을 사용한다.
  • css
/* 태블릿 버전 스타일 (모바일~태블릿 공통 스타일) */
html {font-size: 12px;}
@media screen and (max-width: 767px) { 
    /* 모바일 버전 스타일 (태블릿 대비 모바일에서 달라지는 스타일) */
    html {font-size: 11px;}
}
@media screen and (max-width: 480px) { 
    /* 모바일 버전 스타일2 (모바일 작은 화면에서 달라지는 스타일) */
    html {font-size: 10px;}
}

유의사항

1. 파일 경로 필수 확인후 경로에 맞게 파일 넣기
└ 상대경로를 사용할 때, css 파일 내에서 import 하는 경우와 html의 <head></head> 에서 include 하는 경우 경로가 달라져야 함.
2. mobile.css, mobile.js 인클루드 시 개별 css/js 추가X
3. mobile.css가 아닌 mobile_basic.css 인클루드 시 개별 css 추가 필요
4. 샘플 코드(html)은 필요한 구성요소가 경로에 맞게 들어있는 상태에서 사용. 각 메뉴의 default 에서 필요 파일 확인

파일

1. 모바일 종합 파일 mobile.zip
└ 모바일 요소에서 사용되는 모든 css, js, image 파일 + sample html

01) css
- reset.css
└ 브라우저 기본 스타일 제거 및 기본 설정, 폰트선언 파일
- mobile.css
└ 모바일 요소 css 파일
- mobile_basic.css
└ 모바일 요소중 form과 button 스타일만 들어있는 css 파일
- root.css
└ mobile.css, mobile_basic.css 에서 필요한 변수값을 선언해둔 파일 (pc버전과 동일)
- jquery-ui-1.12.1.custom.css
└ jquery-ui 의 css 파일 (pc버전과 동일)

02) fonts
└ reset.css 에서 필요한 fonts 폴더(pc버전과 동일)

03) images
└ 모바일 요소에 사용되는 img 파일 모음

04) js
- jquery.min.js
└ jqeury 기본 파일 (pc버전과 동일)
- jquery-ui-1.12.1.custom.js
└ jqeury-ui 기본 파일 (pc버전과 동일)
- mobile.js
└ 모바일 요소에 사용되는 script 파일

05) lib
└ 라이브러리 폴더 (xeicon 위치)
