basic.zip 기본요소 종합 파일


기본 파일

유의사항 (필독)

1. 파일 경로 필수 확인후 경로에 맞게 파일 넣기
└ 상대경로를 사용할 때, css 파일 내에서 import 하는 경우와 html의 <head></head> 에서 include 하는 경우 경로가 달라져야 함.
2. 기본 요소 메뉴의 샘플 코드(html)은 basic.zip의 모든 구성요소가 경로에 맞게 들어있는 상태에서 사용.
3. 기본 요소 메뉴의 샘플 코드(css, js)는 basic.css와 basic.js에 모두 선언되어 있으므로 개별 css/js 추가X
└ basic.css, board.css에서 수정해서 사용할 것들은 common.css에 수정 css 적기 (basic, board 수정 X)
└ basic.js에 기본 선언되어있는 스크립트를 수정할땐 basic.js에 직접 수정하되 수정 되었다는 주석 표시
4. UI 요소 에서 사용되는 css, js는 각 메뉴의 샘플코드에서 복사해서 사용. img파일 개별 다운로드 필요. (일괄 파일 없음)
5. css 파일 구조
└ 시스템 사이트의 경우 : 메인+서브에 공통 적용되는 common.css 파일 추가해 메인+서브 공통부분, 서브에 적용되는 css 기재/ 메인에만 적용되는 main.css 추가해 메인 css 기재
└ 일반 사이트의 경우 : 메인+서브에 공통 적용되는 common.css 파일 추가해 메인+서브 공통부분 css 기재 / 메인에만 적용되는 main.css 추가해 메인 css 기재 / 서브에만 적용되는 sub.css 추가해 서브 css 기재
└ 시스템, 일반 사이트 공통 : 멤버십, 에러 등이 필요한 경우 퍼블가이드 UI요소에서 원하는 부분만 복사해서 common.css에 기재

파일

1. 기본요소 종합 파일 basic.zip
└ 기본요소에서 사용되는 모든 css, js, image 파일 + sample html

01) css
- root.css
└ basic.css 에서 필요한 변수값을 선언해둔 파일
- reset.css
└ 브라우저 기본 스타일 제거 및 기본 설정, 폰트선언 파일
- basic.css
└ 기본요소(form & 텍스트 ~ 기타)의 css 파일 (root.css, reset.css import 파일)
- board.css
└ 게시판 요소(게시판 ~ 검색 & 페이징)의 css 파일
- member.css
└ 멤버십 요소의 css 파일
- jquery-ui-1.12.1.custom.css
└ jquery-ui 의 css 파일

02) fonts
└ reset.css 에서 필요한 fonts 폴더

03) images
└ 기본요소, 게시판 요소에 사용되는 img 파일 모음

04) js
- jquery.min.js
└ jqeury 기본 파일
- jquery-ui-1.12.1.custom.js
└ jqeury-ui 기본 파일
- basic.js
└ 기본요소, 게시판 요소에 사용되는 script 파일
- member.js
└ 멤버십 요소에 사용되는 script 파일

05) lib
└ 라이브러리 폴더 (xeicon 위치)
