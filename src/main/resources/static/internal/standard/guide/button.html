기본 설명

1. 버튼의 기본 클래스는 btn이고 사이즈는 sml로 구분한다.
기본 형식: <button type="button" class="btn"></button>
a 버튼 형식: <a href="" class="btn"></a>
2. <button> 태그를 기본으로 하되 기능 상 어쩔 수 없이 필요한 경우에만 <a> 태그를 쓴다.
- 팝업 호출 버튼도 button 태그 사용 (팝업 > 레이어팝업 참고)
3. 버튼은 색상, 크기로 구분한다. 또한 class="bd"을 붙이면 border형 버튼이 된다.
4. 버튼 안에 아이콘이 필요한 경우, https://xpressengine.github.io/XEIcon/library-2.3.3.html의 클래스 이름(xi-?)을 복사해 적용한다.
왼쪽 아이콘: <button type="button" class="btn icl"><i class="xi-?"></i>버튼</button>
오른쪽 아이콘: <button type="button" class="btn icr">버튼<i class="xi-?"></i></button>
5. sidebyside 안의 버튼은 반드시 btn_area로 감싸야 한다.
6. 테이블 안, 테이블 위 버튼은 sml을 사용하고, 테이블 안의 버튼은 btn_area로 감싸지 않는다.

기본 정렬

1. 버튼은 기본 왼쪽 정렬이고, 클래스 옆에 center, right 클래스를 붙여 가운데, 오른쪽 정렬을 한다.
ex) <div class="btn_area center">

코드

  • html
<div class="btn_area left">
    <button type="button" class="btn">좌측 정렬</button>
</div>
<div class="btn_area center">
    <button type="button" class="btn">중앙 정렬</button>
</div>
<div class="btn_area">
    <button type="button" class="btn">우측 정렬</button>
</div>
