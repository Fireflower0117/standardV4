Default

1. UI 요소에서 사용되는 css, js는 각 메뉴의 샘플코드에서 복사해서 추가, img파일 개별 다운로드 후 경로에 맞게 넣어야 함.

메일양식1

1. 문의에 대한 답변 등 컨텐츠가 있는 메일 발송 양식이다.
2. 메일 발송 양식은 웹폰트 (cdn방식, import방식 모두 불가능) 사용이 불가능하므로 각 운영체체 기본 폰트를 사용한다.
3. 메일 발송 양식은 외부 스타일시트를 쓰지 않고 각 태그에 인라인으로 스타일을 적용한다.
4. 이미지 경로, 로고이미지, 하단 회사정보는 각 사이트에 맞게 수정해야한다.

서비스 문의에 대한 답변드립니다.
